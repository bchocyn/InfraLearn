# InfraLearn Armor Set System
## Monster Hunter + Elden Ring Inspired

### Design Philosophy
- **Modular**: Helm, Chest, Gauntlets, Legs separately (not a fixed "top")
- **Elemental**: Each path has an elemental theme + tier progression (Bronze/Silver/Gold)
- **Boss-Inspired**: Elden Ring demigod aesthetic per path
- **Hunterly**: Monster Hunter gear crafting progression feel

## Implementation Approach

### Database Structure (avatarParts.js)
Expand the `top` slot to include armor pieces OR create new `helm`, `chest`, `gauntlets`, `legs` slots:

```javascript
export const AVATAR_ARMOR = {
  fundamentals_bronze: {
    name: 'Stone Warden',
    pieces: {
      helm: 'Stone Crown',
      chest: 'Earthbound Plate',
      gauntlets: 'Stonefist Gloves',
      legs: 'Basalt Leggings',
    },
    unlock: 'fundamentals_33'
  },
  fundamentals_silver: {
    name: 'Granite Sentinel',
    pieces: {...},
    unlock: 'fundamentals_66'
  },
  fundamentals_gold: {
    name: 'Eternal Stone Warden',
    pieces: {...},
    unlock: 'fundamentals_100'
  },
  // ... repeat for all 8 paths × 3 tiers = 24 armor sets
}
```

### PixelLab Generation
Each set can be generated as modular sprites:
```
PixelLab prompt template:
"<tier> <path> armor set piece: <piece_name>, <color theme>, <material description>, 
pixel art game armor, crisp outlines, <element> theme, side view, centered on transparent"

Example:
"Gold tier SWE armor set piece: radiant crown helm, golden-white plate, 
holy light aura, pixel art game armor, radiant celestial theme, 
side view 48x48, centered on transparent"
```

### UI Changes
- New "Armor" tab in AvatarCreator with 8 path buttons
- Within each path, show Bronze → Silver → Gold progression
- Clicking an armor set unlocks/equips all 4 pieces (helm, chest, gauntlets, legs)
- Preview updates in real-time showing full armor set on character

### Progression Feel
- **33% of path**: Bronze gear (starter, practical)
- **66% of path**: Silver gear (upgraded, more ornate)
- **100% of path**: Gold gear (legendary, impossible/ethereal)
- **All 8 paths at 100%**: Unlock a "Master Dragoon" or "Eternal Sentinel" ultimate armor (blend of all 8)

## Monster Hunter Integration Points

1. **Elemental System**: Each path has an element (Earth/Lightning/Data/Holy/Arcane/Dragon/Neutral/Void)
2. **Modular Pieces**: Like MH, armor sets come from path "hunts" (completing lessons)
3. **Tier Progression**: Bronze → Silver → Gold mirrors Low/High/Master rank gear
4. **Crafting Requirement**: "Gather X lessons to unlock Y armor"

---

## Elden Ring Integration Points

1. **Boss-Inspired Design**: Each path's armor looks like it was worn by a demigod of that element
2. **Impossible Aesthetics**: Gold-tier pieces use ethereal effects (glowing, floating, translucent)
3. **Legendary Weapons**: Each path's 100% armor set includes a weapon
4. **Lore Feel**: The armor descriptions weave into the InfraLearn story (Stone Warden = foundation of knowledge, Circuit Warden = infrastructure, Cipher Sage = magical data, etc.)
