REFACTOR OPTIONS · MOCKUP INDEX

InfraLearn dopamine loops.

11 design patterns. Each opens in its own page — click any card. ⭐ marks app-comparison research picks (Brilliant, Sololearn, Zeigarnik, Codecademy).

All-in-one view: refactor-options.html (legacy single-page with horizontal tabs).

01 · Baseline
What users see today
CURRENT STATE · STATIC
8-bullet pros-cons block. Zero interaction. The "wall of dense static" that the research says triggers ADHD skim-bounce.
02 · Click-to-copy chips
Every inline code chip is tappable
STRIPE DOCS · TAILWIND DOCS · S
Tap any code chip → copies + ✓ toast. Reaches every lesson via one renderInline change. Smallest scope, highest reach.
03 · Predict prompts
"What does X output?" before reveal
ROEDIGER & KARPICKE · g≈0.50 · M
Multiple-choice mid-lesson, instant ✓/✗, +5 XP, then explanation. The testing effect — highest-leverage retention mechanic in cog-science.
04 · Tap-to-reveal
Question visible, answer hidden
ANKI · ACTIVE RECALL · S
Curiosity-gap pattern. Show the question, gate the answer behind a tap. Drop-in block type for any glossary-style content.
05 · Live tests
Tests flip green as you type
SANDPACK · FREECODECAMP · L
No "Run" button — feedback is continuous. Watching a test turn green is irresistible. Effort: large per language.
06 · One-tap fix-it
Tap the bug to apply the fix
STACK OVERFLOW CULTURE · M
Buggy code → highlighted bug → tap → it transforms with a satisfying pulse. Makes debugging feel rewarding.
07 · Combo multiplier
3-in-a-row → 1.5× XP, 5 → 2.0×
DUOLINGO · HABIT-LOOP PSYCH · S
Capped at 2× to avoid slot-machine feel. Only ticks on correct answers — learning-anchored. One mistake resets.
08 · Daily Challenge slot
Pinned card on Home · 60-90s
BRILLIANT · USTWO CASE STUDY
One recall question from concepts you already learned. Reuses spaced-rep — zero new content. Top single-feature pick across all 3 research agents.
09 · Fill-in-the-blank
Tap tokens into code, no keyboard
SOLOLEARN · MOBILE-FIRST QUIZ
Active recall without keyboard friction on mobile. Same testing effect, zero typo cost.
10 · Cliffhanger
End each lesson on an open question
ZEIGARNIK · CMU STUDIES
Unresolved questions improve retention AND drive return. The rare social-media-style hook that STRENGTHENS memory.
11 · Runnable inline code
Every code block gets a ▶ Run pill
CODECADEMY · ACM SIGCSE 2023
Removes the "set up Python" cliff. Every static code block becomes a try-it moment. Reuses shipped Pyodide.
RESEARCH VERDICT

Top single-feature pick across all 3 research agents: Tab 08 — Daily Challenge. Cheapest, most ADHD-friendly, most retention-leveraging. Zero new content authoring (reuses spaced-rep store).
