The traps that bite everyone

SAMPLE FROM PY-TYPES LESSON
Lean on this
  • int has arbitrary precision — 2**1000 just works.
  • Strings are immutable, so they're safe as dict keys.
  • f-strings (f"{x=}") are the clearest formatting.
  • bool is an int subclass: sum([True, False, True]).
Don't get burned
  • 0.1 + 0.2 ≠ 0.3 — floats are binary.
  • "5" + 3 raises TypeError.
  • Building strings with += in a loop is O(n²).
  • is checks identity, not equality.
